Open terminal at this location and enter:

'python3 download_CFG.py'

Enter the lectin search term for which all the files are needed. A folder with the lectin name will be created with all the XLS files on the CFG website for that lectin.

The files are named using something called 'primescreen ID' and more information about the data can be extracted using the following link:

http://www.functionalglycomics.org/glycomics/HServlet?operation=view&sideMenu=no&psId= <enter filename here>
